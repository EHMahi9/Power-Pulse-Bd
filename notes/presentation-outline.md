# Presentation Outline

## 1. Problem

Bangladesh users often face local power interruptions without a simple community-level view. Many families also own IPS or solar backup systems but cannot easily estimate how long their battery will last.

## 2. Proposed Solution

PowerPulse BD provides:

- outage reporting,
- community feed,
- dashboard metrics,
- solar backup estimator,
- admin verification.

## 3. Syllabus Connection

- HTML/CSS: responsive interface.
- JavaScript: forms, DOM events, routing.
- Backend: Node server and REST APIs.
- Database: local JSON persistence with relational mapping.
- Security: password hashing, token auth, route protection.
- OOP: solar backup engine using classes and polymorphism.

## 4. Demo Flow

1. Login with the demo account.
2. View dashboard metrics.
3. Submit an outage report.
4. Open community feed and filter by district.
5. Run the solar backup estimator.
6. Login as admin and verify a report.

## 5. Future Scope

- React + Vite frontend.
- PostgreSQL + Prisma.
- Live Bangladesh map.
- Multi-user confirmation scoring.
- Bangla UI.
- SMS/email alerts.
- Official utility notice integration.
